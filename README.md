Calculating precise fasting times across different cities requires complex lunar and timezone math. This project bypasses that complexity by fetching real-time, astronomically accurate data from the Aladhan REST API. It strips away heavy frameworks to provide a clean, instant-loading UI that works perfectly on mobile browsers.
